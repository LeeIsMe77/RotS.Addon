Return of the Shadow Addon Module

Hello. I'm Lee Gurley and this is my add-on repository for COM DLL modules intended to be consumed by the Jaba Mud Client (JMC) for use with the Multi-User Dimension (MUD) text-based game Return of the Shadow.

I'll be updating this page incrementally to try and keep you up to date with any chances as they occur with this module, and as I get used to using GitHub Pages. Stay tuned for more updates!
