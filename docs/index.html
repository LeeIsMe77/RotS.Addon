Return of the Shadow Addon Module

Hello. I'm Lee Gurley and this is my add-on repository for COM DLL modules intended to be consumed by the Jaba Mud Client (JMC) for use with the Multi-User Dimension (MUD) text-based game Return of the Shadow . These addons are written in C# and are designed to introduce quality of life changes to the game by automating simple tasks. Each module is designed to contain custom settings to make them nearly 100% user customizable.

This repository began with my realization that after writing 1200 lines of JScript, I can actually load up ActiveX COM components into JMC Mud client. At this point, it was time for me to start researching COM and determine how I could create a well-structured project that made the importing of all the JScript code feasible. Once I got the core library complete, creating a few sample modules hardly took any time at all.

Now that I've gotten the basics out of the way, I'm incredibly excited to continue porting the JScript code into C# modules while adding some extra functionality that was just not at all feasible using pure JScript. I'll be updating this page incrementally to try and keep you up to date with any changes as they occur with this project, and as I get used to using GitHub Pages. If anyone has any requests for features they would love to see implemented, I'd absolutely love to hear them. Stay tuned for more updates!
